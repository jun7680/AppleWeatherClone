# AppleWeatherClone

### Stack
1. MVVM
2. Alamofire, RxDataSource, RxSwift, Snapkit

## 기능구현
### Main
1. SearchBar 클릭시 SeachViewController present
2. 현재 날씨의 최고, 최저 온도, 현재 온도, 현재 날씨 상태
3. 3시간별 최대 2일까지의 온도
4. 5일간 최대 온도, 최저 온도
5. 현재 검색한 위치
6. 현재 날씨의 습도, 구름양, 바람속도

### 검색
1. 프로젝트의 citylist.json을 파싱하여 리스트 표시
2. 리스트 선택시 Main화면에서 해당 지역 날씨 표시
